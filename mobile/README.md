# MikroServer Mobile (Cross-Platform)

This project is now structured as a single Expo app for:
- Android
- iOS
- Web

## Commands

```bash
npm install
npm run android
npm run android:reinstall
npm run android:prod
npm run ios
npm run web
```

## Notes

- Android native sources are kept in `android/` and can still be built with Gradle.
- iOS sources are generated by Expo prebuild when needed (`npx expo prebuild -p ios` on macOS).
- Shared app routes and UI are in `app/`.
- The mobile app now loads your existing web app in an embedded WebView (full web feature parity in mobile).
- Web URL is configurable directly inside the app (`URL` button in header) and persisted.
- You can also set a default URL at build time with `EXPO_PUBLIC_WEB_APP_URL`.
- `npm run android:prod` builds a release APK from a short path (`C:\dev\mikroserver-mobile-build`) and installs it on a connected Android device via `adb`.
- `npm run android:reinstall` keeps a debug flow for local development.
- Built artifacts are copied to `artifacts/app-release.apk` and `artifacts/app-debug.apk`.
