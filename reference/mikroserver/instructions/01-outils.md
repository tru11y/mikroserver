# ✅ ÉTAPE 1 — Vérifier que ton PC est prêt

Ouvre **PowerShell** et tape ces commandes une par une.
Si tu vois un numéro de version → c'est installé ✅
Si tu vois "not recognized" → c'est pas installé ❌

---

## Vérification Node.js

Tape dans PowerShell :
```
node --version
```
Tu dois voir quelque chose comme : `v20.0.0` ou plus ✅

---

## Vérification npm

```
npm --version
```
Tu dois voir quelque chose comme : `10.0.0` ou plus ✅

---

## Vérification Docker

```
docker --version
```
Tu dois voir quelque chose comme : `Docker version 24.0.0` ✅

> ⚠️ Docker Desktop doit être **ouvert et en marche** (cherche la baleine 🐋 dans la barre des tâches en bas à droite)

---
